## About

**`Hyper Fetch CLI`** is our command line featuring request sdk generators.

## Features

‚ú® **Generate code** - from **OpenApi V3** / **Swagger** schema - both types and actual requests.

‚ú® **Flexible** - after generation you can edit Hyper Fetch client and all related requests.

‚ú® **No additional boilerplate** - if you know Hyper Fetch, you don't need anything else.

```tsx
npx @hyper-fetch/cli@latest
```

...and it works! All code generated and ready to use.
