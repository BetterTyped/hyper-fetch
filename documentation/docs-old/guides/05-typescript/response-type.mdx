# Response Type

Response type defines the data that will be returned to us from the server

---

### Example

```ts
type ResponseType = Array<{
  name: string;
  email: string;
  age: number;
}>;

const getUsers = client.createRequest<{response: ResponseType}>()({
  endpoint: "/users",
});
```
