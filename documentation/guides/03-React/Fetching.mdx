## useFetch

For fetching data use the `useFetch` hook that automates command handling. There are tons of options to adjust this hook
so check out the documentation.

```tsx
import { getUsers, getUser } from "server/user";

...

const { data, error, isLoading } = useFetch(getUsers)

const { data, error, isLoading } = useFetch(getUser.setParams({ userId: 1 }))


...


```
