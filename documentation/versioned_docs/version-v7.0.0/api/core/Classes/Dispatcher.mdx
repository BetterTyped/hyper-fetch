  # Dispatcher
  
  <div class="api-docs__description ">
    
    Dispatcher controls and manages the requests that are going to be executed with adapter. It manages them based on the options provided with request.
    This class can also run them with different modes like deduplication, cancelation, queueing or run-all-at-once mode. With it&#x27;s help we can pause,
    stop, start and cancel requests.
