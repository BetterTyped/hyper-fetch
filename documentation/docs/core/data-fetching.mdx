# Fetching

> Fetching can be done by using [Request](https://hyperfetch.bettertyped.com/docs/guides/basic/setup/#create-request)
> and [useFetch](https://hyperfetch.bettertyped.com/docs/documentation/react/core/use-fetch/) hook.

---

It super easy to fetch the data using Hyper Fetch.

```tsx
import { getUsers, getUser } from "./requests";

// ...

const App = () => {
  const { data, error, loading } = useFetch(getUsers);
  return (
    <div>
      {loading && <div>Loading...</div>}
      {error && <div>{error.message}</div>}
      {data && <div>{JSON.stringify(data)}</div>}
    </div>
  );
};
```

##### Read more about [useFetch and fetching](https://hyperfetch.bettertyped.com/docs/documentation/react/core/use-fetch/) on Hyper Fetch documentation.
