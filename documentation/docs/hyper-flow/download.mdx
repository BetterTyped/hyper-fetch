# Download

HyperFlow is a cross-platform application available for macOS, Windows, and Linux. Below you can find the appropriate
installation package for your operating system.

---

## macOS

For macOS, we provide a `.dmg` file that allows for easy installation. We also offer a `.zip` archive for manual
installation. HyperFlow is optimized for both Intel and Apple Silicon architectures.

[Download HyperFlow.dmg](https://github.com/BetterTyped/hyper-fetch/releases?q=hyper-flow&expanded=true)

---

## Windows

For Windows, we offer a straightforward `.exe` installer that will guide you through the setup process.

[Download HyperFlow](https://github.com/BetterTyped/hyper-fetch/releases?q=hyper-flow&expanded=true)

---

## Linux

For Linux, we provide both `.deb` (for Debian/Ubuntu-based distributions) and `.rpm` (for Fedora/CentOS-based
distributions) packages.

[Download HyperFlow_amd64.deb](https://github.com/BetterTyped/hyper-fetch/releases?q=hyper-flow&expanded=true)

[Download HyperFlow-x86_64.rpm](https://github.com/BetterTyped/hyper-fetch/releases?q=hyper-flow&expanded=true)
